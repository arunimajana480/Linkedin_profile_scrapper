

# LinkedIn Profile Scraper in Python

This is a Python script that allows users to scrape public LinkedIn profiles and extract relevant information such as name, job title, education, and work experience.

## Features

- **Profile Scraper**: The script allows users to input a LinkedIn profile URL and extract relevant information.

- **Customization**: The script allows users to choose which information to extract and how to format the output.

- **Error Handling**: The script includes error handling to ensure that the user inputs a valid LinkedIn profile URL.

## Installation

1. Clone the repository to your local machine.

2. Install the required dependencies using pip.

```
pip install -r requirements.txt
```

3. Create a [LinkedIn developer account](https://www.linkedin.com/developers/) and obtain an API key.

4. Update the values in the `config.ini` file with your API key and secret.

## Usage

1. Run the script using Python.

```
python Profile_Scrapper.py
```

2. Follow the prompts to input the LinkedIn profile URL and choose which information to extract.

3. The script will output the extracted information in the specified format.
